# House Price Prediction

## Objective
Predict house prices based on area and number of bedrooms.

## What this project does
- Loads housing data
- Trains a Linear Regression model
- Predicts house prices based on input values

## Steps performed
1. Imported required libraries (pandas, sklearn)
2. Loaded dataset
3. Selected features (area, bedrooms)
4. Split data into training and testing
5. Trained model using Linear Regression
6. Made predictions

Area = 1500  
Bedrooms = 3
